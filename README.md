# Algorithmic Bias in Boston Crime Data

This project explores how algorithmic systems can shape the way we interpret crime data in Boston neighborhoods. Instead of presenting data as purely objective, this site highlights how bias can be introduced through data collection, categorization, and analysis.

---

## Project Overview

The website is designed as a **public-facing interactive experience** that allows users to explore crime data visually while learning how algorithmic bias works behind the scenes.

### Key Features
- 🗺️ Interactive map of Boston police districts  
- 📊 Data visualizations showing crime distribution  
- 🔍 Explanations of how bias can appear in datasets  
- 🎯 Reflection section to encourage critical thinking  

---

## Purpose

The goal of this project is to make complex ideas like **algorithmic bias** more accessible to a general audience. By combining simple explanations with interactive visuals, users can better understand how data-driven systems are not always neutral.
